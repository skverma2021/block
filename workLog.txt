Project Summary: Blockchain PoC (Proof of Concept)
This is a private blockchain network implementation for environmental monitoring data, built with Node.js/Express and SQLite.

Architecture
Multi-node network with role-based nodes:
RegAuth (Node 0): Regulatory authority that mines blocks
Project Nodes (1, 2, etc.): Submit and receive transactions/blocks

Core Features
Component	        Purpose
db.js:              SQLite persistence with tables for blockchain, mempool, and confirmed transactions
transactions.js:    Submit new transactions, receive broadcasts, hash validation
blocks.js:          Mining logic with Merkle tree, block creation, and broadcasting
network.js:         Peer discovery, node registration, network topology management

Key Characteristics
Proof of Authority (PoA) consensus - only RegAuth mines blocks
Auto-mining triggers when mempool reaches 5 transactions (every 10 seconds check)
Transaction data: Environmental readings (SO2, NO2, PM10, PM2.5) from monitoring stations
Integrity: SHA-256 hashing on transactions and blocks, Merkle root calculation
Sync: Non-RegAuth nodes sync the full chain from RegAuth on startup

Tech Stack
Express 5.1, SQLite3, Axios (for node communication), UUID, SHA256, CORS enabled
